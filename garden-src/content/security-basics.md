# Security Basics

Information security is about protecting information and systems from unauthorized access, misuse, disruption, modification, or destruction.

A useful beginner model is the **CIA triad**:

| Principle | Meaning | Example |
|---|---|---|
| Confidentiality | Only authorized people can access the information | Using encryption or access controls |
| Integrity | Information stays accurate and unaltered | Detecting unauthorized file changes |
| Availability | Systems and data are accessible when needed | Backups, redundancy, and DDoS protection |

## Authentication vs. authorization

**Authentication** asks: “Who are you?”

**Authorization** asks: “What are you allowed to do?”

For example, logging into an account is authentication. Whether you can view a specific file, gradebook, or admin panel is authorization.

## Common security habits

- Use a password manager.
- Turn on multifactor authentication.
- Keep devices and browsers updated.
- Be cautious with links and attachments.
- Back up important files.
- Review app permissions.
- Avoid reusing passwords.
