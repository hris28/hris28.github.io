# Welcome to the garden

This is my working space. It's Obsidian compatible markdown, meaning you can drop
these files into any Obsidian vault and they will behave normally, wiki links and all.

## What lives here

A mix of things that are less formal than a blog post and less finished than a paper.
Reading notes, working hypotheses, half-formed arguments, code snippets I want to
remember. Some notes will stay stubby. Others will grow.

## Where this is going

The longer-term plan is an OSINT themed Information Security guide, built up note by
note over my graduate studies. Start here: [[osint-intro|OSINT, an orientation]].

## How to use this page

Search the top bar by keyword, or filter by tag. Click a note to open it. Notes link to
each other with wiki links like [[hci-eye-tracking]], which route through the same viewer.

> A garden, not a stream. Things here get tended to, not published once.
